六、你必须重点回答的短线问题
==============================
1. 当前热点催化到底是什么？
- 是公司公告、行业政策、产业催化、业绩催化、订单催化、事件催化，还是纯市场情绪？
- 是否存在明确日期、明确文件、明确公告？
- 催化是 A 级硬催化、B 级半硬催化，还是 C 级弱关联/情绪催化？

2. 当前题材是否处在可交易阶段？
- 发酵初期 / 扩散期 / 加速期 / 高位分歧 / 退潮期
- 板块是单点异动，还是出现了明显扩散和前排梯队？
- 未来 1~5 个交易日是否还有继续发酵的时间窗口？

3. 这只股票在题材中的位置是什么？
- 龙头 / 前排 / 中军 / 补涨 / 跟风 / 纯套利
- 它的辨识度来自连板高度、容量、成交额、公告硬度，还是纯盘口情绪？
- 与同题材 2~4 个核心股相比，它的位置到底如何？

4. 公司与题材的相关性是真实还是弱关联？
- 公司主营、产品、订单、客户、收入占比、公告表述，是否能支撑它被归入该题材？
- 它是直接受益、间接受益，还是概念擦边？
- 市场是否在"先炒想象、后看兑现"？

5. 第一阶段量价强势信号背后意味着什么？
- 涨停次数、龙虎榜次数、换手率、成交额放大，到底代表主线确认，还是高位拥挤？
- 一字板、长上影、炸板 proxy，是否说明筹码不稳或接力难度很大？
- 高换手是健康换手，还是高位筹码松动？

6. 当前是否具备可交易性？
- 次日大概率是一字不可参与，还是有分歧换手机会？
- 更适合观察"高开弱转强 / 分歧承接 / 放量突破 / 低开修复"，还是不适合参与？
- 是否已经过度偏离均线与合理承接区间？

7. 短期硬风险是否在上升？
- 减持、解禁、质押、异动监管、问询、业绩不及预期、题材证伪、公告澄清
- 是否存在"明天或未来几天就可能打断交易逻辑"的事件？
- 第一阶段的股东减持、解禁、质押等 flags，到底只是背景噪音，还是短线致命变量？

8. 次日到未来5日，延续概率如何？
- 为什么可能继续强？
- 为什么可能次日就结束？
- 市场为什么可能是对的？
- 这只票为什么可能并不值得短线继续跟踪？
